**Photon AI — жасанды интеллект көмегімен презентацияларды (PDF және PPTX) автоматты түрде 2-3 минут ішінде жасауға арналған дайын SaaS құралы.**

### ✨ Негізгі мүмкіндіктері:

  - Ақылды құрылым: ЖИ (Groq Compound) іш пыстыратын тізімдерді емес, логикалық блоктарды, тақырыптарды және нақты фактілерді жасайды.
  
  - Typography-first дизайн: Қатаң блоктық құрылым, мінсіз шегіністер және қымбат журнал стилін жасау үшін Roboto қаріпінің бірнеше нұсқалары қолданылады.
  
  - Көптілділік: Қазақ, орыс және ағылшын тілдеріндегі презентацияларды жасау.
  
  - Дизайн-жүйесі: Кіріктірілген түстер палитрасы (Corporate Blue, Creative Orange, Dark Mode, Green Eco) және бейімделген макеттер (Торлар, Карточкалар, Ортаға туралау).
  
  - Жылдам экспорт: Серверде уақытша файлдарды сақтамай-ақ, дайын .pdf немесе .pptx файлдарын бірнеше секунд ішінде жүктеп алу.

 ### ⚒ Технологиялық стек:
 
  - Frontend: Таза HTML/CSS/JS (iOS стиліндегі UI).
 
  - Backend: FastAPI (асинхронды оркестратор).
  
  - AI Engine: Groq API (compound моделі арқылы жылдам генерация).
  
  - Export Engine: reportlab (векторлық PDF) және python-pptx.

 photon-ai/
